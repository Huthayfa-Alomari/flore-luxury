# 3D Models for AR

Add your .glb/.gltf 3D models here for AR preview.

Expected files (matching seed data):
- blush-velvet.glb
- golden-amber.glb
- royal-peony.glb
- crystal-vase.glb
- eternal-dome.glb
- marble-vase.glb

If models are not available, the app will gracefully fall back to image previews.

You can generate 3D models using:
- Blender (free)
- Polycam (mobile scanning)
- Sketchfab (purchase models)
